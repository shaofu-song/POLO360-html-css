# POLO360-html-css
  ### POLO360网站首页布局
* 项目描述
  * 该项目是将一个 PSD 设计图转换为一个商业网站的首页，内容涵盖HTML基础、标签、CSS 选择器、盒子模型、浮动、定位、图片整合、PS 切图等页面相关常用技术。熟练掌握 PC 端的固定布局。
# html-css
  ### oppo主页
* 项目介绍：
  * 该项目实现oppo主页的纯页面布局。
* 实现技术：
  * 1. 利用浮动和层定位原理实现分栏布局。
  * 2. 利用swiper组件实现轮播效果。
  * 3. 利用animate动画库实现弹入弹出动画。
  * 4. 利用JS的DOM操作onscroll实现返回顶部和呼吸灯效果。
